// installation

Up and running in 3 steps

No Python setup, no terminal fiddling. Just download, install, and go.

1

Download the Installer

Click the button below to download LdragoSetup.exe — a self-contained Windows installer (~few MB).

LdragoSetup.exe ← click Download above
2

Run the Installer

Double-click the downloaded file. If Windows SmartScreen appears, click "More info → Run anyway" — the app is safe but unsigned.

[sys] Launched my_agent.exe [sys] Workspace root: D:\agent-0\quantum\agentic\Ldrago
3

Start Chatting

Ldrago loads the local model and shows [ok] --- System Ready ---. Type your first message and press Send.

[ok] --- LdragodestructorAI Agentic System Ready (Type 'exit' to quit) ---
